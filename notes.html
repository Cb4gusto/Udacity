HTML UDACITY NOTES

Lesson 1: Basics of the Web

World Wide Web

WWW made up of numberous HTML pages (Hyper Text Markup Language) which is also the main type of document on the web.

Components of Web

User => Web Browser => Internet => Servers and back. You navigate a webpage in turn your browser makes requests via the Internet to Servers. The request uses HTTP protocol in which the server responds by sending back files that the browser displays

Lesson 2: Basics of HTML Tags

Markup

Markup is how text would look, we manipulate this using elements, these elements are made up of an opening tag, content in the middle and a closing tag.
Make sure to have a closing tag for those that need it or else the formating will apply to everything after its opening tag.

Basic Tags

Bold, Italic, Links, Paragraph, Break/New Line, Images, Inline v Block/Box Elements, Boxes & Containers, Span & Div, Document Structure
< b > Bolded < /b > | < em > Italicized < /em > | < anchor attribute="value/url" > words to show < /anchor > | < p > paragraph < /p > | < br >
Bolded | Italicized | Bolded AND Italicized |A Programmers Best Link | <== All Inline elements

Paragraph tags are box elememts as illustrated by using a border as well as invisible top ^ and bottom v margins that create space between other elements

< br > tag makes a new line that is invisible, also a void tag

Don't forget about images as well with < img src="url" alt="text" > (img is a void tag, doesn't need closing tag)

Images are also inline elements

< span > < /span > tags create an inline container element, while < div > < /div > tags create a box container element
